DJ Visualizer Help

Quick guide for loading music and controlling visuals.

English

  • Press O to choose a music folder.
  • Press R to rescan after adding files.
  • Drag and drop files to add extra tracks.
  • AIFF playback works best in Safari.
  • Total playlist time + loop count are shown in the player.

Shortcuts

  • Space Play / Pause
  • ← / → Previous / Next
  • 1-9, 0 Visual modes
  • C Color scheme
  • O Folder picker
  • R Rescan folder
  • H Help
  • F Fullscreen

日本語

  • O でフォルダを選択。
  • R で再スキャン（曲追加後）。
  • ドラッグ＆ドロップで曲を追加。
  • AIFF は Safari 推奨。
  • 合計時間とループ回数が表示されます。
